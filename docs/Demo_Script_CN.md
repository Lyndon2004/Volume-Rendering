# 海洋含氧量体渲染可视化 - 演示稿（1-2 分钟）

## 📋 演示概述

**主题**：海洋含氧量三维分布可视化系统

**数据**：
- 400×441×92 分辨率的海洋含氧量数据（单日快照）
- 数据范围：0-203 mg/L（已归一化到 0-255）

**技术**：
- Raymarching 体渲染（VolumeSTCube 框架）
- 语义化 Transfer Function（颜色编码含义）
- 内部沉浸式观察模式

---

## 🎨 Transfer Function 设计（颜色编码）

### 色彩语义映射

```
密度值范围    →    颜色          →    含氧量含义
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
0.0 - 0.2    →    红色          →    缺氧危险区（需要关注）
0.2 - 0.4    →    橙色          →    低氧警告区
0.4 - 0.6    →    绿色/绿蓝色    →    适氧区（正常水平）← 主要关注区
0.6 - 0.8    →    浅蓝色        →    高氧区（良好）
0.8 - 1.0    →    深蓝色        →    极高氧区（优异）
```

### 透明度策略

- **缺氧区**（密度 0.1-0.2）：高透明度（α=0.9）→ **强调、立即可见**
- **适氧区**（密度 0.4-0.6）：低透明度（α=0.4）→ **背景淡化（非重点）**
- **高氧区**（密度 0.6-1.0）：中等透明度（α=0.5-0.6）→ **点缀、展示良好**

**设计理念**：用颜色直观表达物理意义，用透明度突出关键区域（缺氧）

## ⚙️ 当前材质参数设置

| 参数 | 值 | 含义 |
|------|-----|------|
| **BaseDensityAlpha** | 1.0 | 平坦区域完全保留，不淡化 |
| **InternalDensityScale** | 0.1~1.0 | 内部观察的密度累积（可根据需要调） |
| **EdgeContribution** | 0 | 不强调梯度边界（海洋数据无科学意义） |
| **MinGradient** | 0.01 | 梯度可见性阈值（不影响 Edge=0） |
| **RaymarchSteps** | 512+ | 采样精度（可提高到 1024） |

## 📊 演示流程（1-2 分钟）

### 第 1 段（0:00-0:30）：外部观察

**相机位置**：在体积外部（距离中心约 2-3 倍）

**操作**：
1. 旋转相机，展示整个海洋体积的三维结构
2. **关键点**：
   - 展示边界的颜色变化（中心绿→边缘红蓝）
   - 强调缺氧区（红色）的位置与分布
   - 演示适氧区（绿蓝色）作为主体背景

**解说词**：
> "这是某海域一天的含氧量三维分布。红色代表缺氧区（0-20%），需要特别关注；绿蓝色代表适氧区（40-60%），是海洋的主体；蓝色代表高氧区（60-100%），表示含氧量充足。通过体渲染，我们可以一目了然地看到各个区域的空间分布。"

---

### 第 2 段（0:30-1:00）：缩放进入

**操作**：
1. 缓慢缩放相机向中心靠近
2. 穿过表面，进入体积内部

**关键点**：
- 外部看时边界轮廓清晰
- 内部看时，缺氧、适氧、高氧层次分明
- 颜色平滑过渡，没有突变

**解说词**：
> "缩放进入后，我们进入到海洋内部。可以看到，各个含氧量层次按深度分布。缺氧危险带（红色）多分布在特定深度，而不是均匀分布——这正是我们想要观察和分析的。"

---

### 第 3 段（1:00-1:30）：内部观察与交互

**相机位置**：在体积内部

**操作**：
1. 在内部自由旋转和移动，观察 3D 层次
2. 如有时间，用鼠标激光钻功能"挖掘"某个区域（如果已实现）

**关键点**：
- 展示内部的"分层"结构
- 缺氧区的集中度与位置
- 颜色的连续性和平滑过渡

**解说词**：
> "从内部观察，我们可以发现含氧量的真实分布模式。缺氧区不是随意分布，而是在特定的深度和位置集中。这样的三维可视化让海洋学家能够直观理解复杂的物理过程。"

---

**如果有时间和技术支持**：

   - 展示不同的 TF 预设（语义 vs 通用 vs 高对比）
   - 说明为什么语义 TF 更适合海洋数据

2. **参数调整**（如有 UI）：
   - 调整透明度参数，展示对可视化的影响

**解说词**：

## 📌 关键数据与参数速查表

### 当前使用的数据
```
文件: volume_oxygen_data_time_0_255.raw
维度: 400 × 441 × 92
范围: 0-203（原始单位）或 0-1（归一化）
格式: uint8
```

### Transfer Function 控制点
```
色彩点:
  0.0   → RGB(0.1, 0.0, 0.0)      // 深红
  0.15  → RGB(0.9, 0.15, 0.0)     // 鲜红
  0.35  → RGB(0.95, 0.6, 0.1)     // 橙色
  0.5   → RGB(0.3, 0.75, 0.35)    // 绿色
  0.65  → RGB(0.2, 0.6, 0.85)     // 浅蓝
  0.85  → RGB(0.15, 0.35, 0.95)   // 深蓝
  1.0   → RGB(0.1, 0.2, 0.9)      // 极深蓝

Alpha 点:
  0.0   → 0.0   // 无数据不显示
  0.08  → 0.85  // 缺氧区：高可见
  0.2   → 0.95  // 低氧区：极高可见
  0.5   → 0.4   // 适氧区：淡化背景
  0.85  → 0.6   // 高氧区：可见
  1.0   → 0.7   // 极高氧：稳定显示
```

### 材质参数
```
BaseDensityAlpha: 1.0
InternalDensityScale: 0.6-1.0（根据内外观察调）
EdgeContribution: 0（不用）
MinGradient: 0.01
```
